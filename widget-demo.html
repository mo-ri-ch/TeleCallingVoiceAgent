This is a dummy host page — it does NOT know anything about the AI widget below.

This static page simulates a client's website. The only thing added to embed the AI Telecaller agent is a single <script> tag at the bottom of the page — nothing else on this page changes.

Open this file directly in a browser. You should see a floating chat launcher button appear in the bottom-right corner. Click it to open the chat panel, type a question, or click the microphone icon to ask a question by voice.

All of the widget's styles and markup live inside a Shadow DOM, so they cannot clash with (or be affected by) this host page's CSS.
